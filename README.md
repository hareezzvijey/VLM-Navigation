# VLM Navigation Pipeline

## Overview
This project uses:
- Grounding DINO (object detection)
- SAM (segmentation)
- Multi-pass prompting
- Scene understanding

## Features
- Detects people, vehicles, and path
- Filters irrelevant objects
- Estimates distance
- Generates scene descriptions

## Example Output
"Person ahead. Sidewalk available."
